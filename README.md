# Trimalaksana ERP

A cross-platform desktop ERP application built with Electron, Node.js, and TypeScript.

## Features

- Full ERP functionality for business management
- Cross-platform: runs on Windows, macOS, and Linux
- Modern UI built with TypeScript and TSX
- Local-first desktop experience

## Tech Stack

- **Runtime:** Electron
- **Backend:** Node.js
- **Language:** TypeScript / TSX
- **Platform:** Windows · macOS · Linux

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
# Windows
npm run build:win

# macOS
npm run build:mac

# Linux
npm run build:linux
```
