## Pengaturan & Konfigurasi

### Profil Pengguna

#### Memperbarui Profil

1. Klik **Ikon Profil** (kanan atas)
2. Klik **Profil Saya**
3. Edit detail:
   - Nama
   - Email
   - Telepon
   - Departemen
4. Klik **Simpan**

#### Mengubah Password

1. Pergi ke **Pengaturan** → **Ubah Password**
2. Masukkan password saat ini
3. Masukkan password baru
4. Konfirmasi password baru
5. Klik **Simpan**

### Preferensi

#### Mengatur Preferensi

1. Pergi ke **Pengaturan** → **Preferensi**
2. Konfigurasi:
   - **Bahasa**: Bahasa sistem
   - **Format Tanggal**: DD/MM/YYYY atau MM/DD/YYYY
   - **Mata Uang**: Mata uang tampilan
   - **Zona Waktu**: Zona waktu Anda
   - **Notifikasi**: Peringatan email aktif/nonaktif
3. Klik **Simpan**

### Pengaturan Perusahaan

#### Memperbarui Info Perusahaan

1. Pergi ke **Pengaturan** → **Pengaturan Perusahaan** (hanya Admin)
2. Edit:
   - Nama perusahaan
   - Alamat
   - Telepon
   - Email
   - Logo
   - NPWP
3. Klik **Simpan**

## Pemecahan Masalah

### Masalah Umum

#### Masalah: Tidak Bisa Login

**Solusi**:
1. Periksa username dan password benar
2. Verifikasi CAPS LOCK mati
3. Jika akun terkunci, hubungi administrator
4. Coba hapus cache dan cookie browser
5. Coba browser berbeda

#### Masalah: Data Tidak Tersinkronisasi

**Solusi**:
1. Periksa koneksi internet
2. Segarkan halaman (F5)
3. Periksa apakah server online
4. Coba logout dan login kembali
5. Hubungi dukungan IT jika berlanjut

#### Masalah: Performa Lambat

**Solusi**:
1. Tutup tab browser lain
2. Hapus cache browser
3. Kurangi jumlah baris yang ditampilkan
4. Gunakan filter untuk mempersempit data
5. Coba browser berbeda
6. Periksa kecepatan internet

#### Masalah: Tidak Bisa Cetak/Ekspor

**Solusi**:
1. Periksa pengaturan cetak browser
2. Pastikan pembaca PDF terinstal
3. Coba browser berbeda
4. Periksa izin file
5. Hubungi dukungan IT

#### Masalah: Pekerjaan Tidak Tersimpan Hilang

**Solusi**:
1. Sesi habis setelah 30 menit tidak aktif
2. Selalu klik **Simpan** sebelum meninggalkan halaman
3. Hati-hati menggunakan tombol kembali browser
4. Hubungi administrator jika data hilang

## FAQ (Pertanyaan yang Sering Diajukan)

### Pertanyaan Umum

**T: Bisakah saya mengakses sistem dari ponsel saya?**  
J: Ya, sistem responsif dan bekerja di tablet dan ponsel. Untuk pengalaman terbaik, gunakan tablet atau desktop.

**T: Bagaimana jika saya lupa password?**  
J: Klik "Lupa Password" di halaman login. Anda akan menerima email dengan tautan reset.

**T: Bisakah saya bekerja offline?**  
J: Kemampuan offline terbatas. Beberapa fitur memerlukan koneksi internet. Perubahan tersinkronisasi saat Anda terhubung kembali.

**T: Seberapa sering data dicadangkan?**  
J: Data dicadangkan secara otomatis setiap jam. Hubungi IT untuk detail cadangan.

### Pesanan & Penjualan

**T: Bisakah saya mengedit pesanan penjualan yang dikonfirmasi?**  
J: Tidak, pesanan yang dikonfirmasi tidak dapat diedit. Anda harus membatalkan dan membuat yang baru.

**T: Apa yang terjadi jika pelanggan membatalkan pesanan?**  
J: Hubungi manajer Anda. Proses pembatalan tergantung pada status pesanan.

**T: Bisakah saya membuat pengiriman sebagian?**  
J: Ya, Anda dapat mengirimkan sebagian pesanan dan menyelesaikan sisanya nanti.

### Inventaris & Stok

**T: Bagaimana stok dihitung?**  
J: Stok = Stok Awal + Pembelian - Penjualan - Pengembalian

**T: Apa itu level pemesanan ulang?**  
J: Level stok minimum. Sistem memberi peringatan saat stok jatuh di bawah ini.

**T: Bisakah saya menyesuaikan stok secara manual?**  
J: Hanya administrator yang dapat menyesuaikan stok. Hubungi manajer Anda.

### Keuangan & Pembayaran

**T: Kapan saya harus mencatat pembayaran?**  
J: Catat pembayaran saat Anda benar-benar menerima uang dari pelanggan.

**T: Bisakah saya menghapus faktur?**  
J: Tidak, faktur tidak dapat dihapus. Anda dapat membatalkan jika diperlukan.

**T: Metode pembayaran apa yang didukung?**  
J: Tunai, Transfer Bank, Cek, Kartu Kredit (jika dikonfigurasi).

### Teknis

**T: Browser apa yang didukung?**  
J: Chrome, Firefox, Safari, Edge (versi terbaru disarankan).

**T: Apakah data saya aman?**  
J: Ya, semua data dienkripsi dan dicadangkan. Akses berbasis peran.

**T: Bisakah saya mengekspor semua data saya?**  
J: Ya, pergi ke Pengaturan → Ekspor Data. Hubungi IT untuk ekspor massal.

### Produksi (Packaging)

**T: Apa itu SPK?**  
J: SPK (Surat Perintah Kerja) adalah dokumen yang memberitahu tim produksi apa yang harus diproduksi.

**T: Bagaimana jika produksi gagal QC?**  
J: Produk ditolak dan SPK kembali ke produksi untuk diperbaiki.

**T: Bisakah saya mengubah tanggal produksi?**  
J: Ya, selama SPK masih dalam status DRAFT. Hubungi manajer untuk perubahan lainnya.

### Pengiriman (Trucking)

**T: Bagaimana cara melacak pengiriman?**  
J: Buka Pesanan Pengiriman dan lihat status real-time dan lokasi pengemudi.

**T: Apa itu Surat Jalan?**  
J: Dokumen pengiriman yang diberikan kepada pengemudi sebelum keberangkatan.

**T: Bagaimana jika pengiriman terlambat?**  
J: Hubungi pengemudi atau manajer logistik untuk informasi.

## Tips & Trik

### Menghemat Waktu

1. **Gunakan Pencarian**: Cari produk, pelanggan, atau pesanan dengan cepat
2. **Gunakan Filter**: Saring data untuk menemukan apa yang Anda butuhkan
3. **Gunakan Shortcut Keyboard**: 
   - Ctrl+S: Simpan
   - Ctrl+P: Cetak
   - Ctrl+F: Cari
4. **Bookmark Halaman Favorit**: Akses cepat ke halaman yang sering digunakan

### Menghindari Kesalahan

1. **Selalu Tinjau Sebelum Konfirmasi**: Periksa detail sebelum mengkonfirmasi
2. **Gunakan Daftar Periksa**: Buat daftar untuk memastikan tidak ada yang terlewat
3. **Simpan Secara Berkala**: Jangan menunggu sampai akhir untuk menyimpan
4. **Gunakan Catatan**: Tambahkan catatan untuk referensi di masa depan

### Meningkatkan Produktivitas

1. **Gunakan Template**: Buat template untuk dokumen yang sering digunakan
2. **Otomatisasi Tugas**: Gunakan fitur otomatis sistem
3. **Delegasi Tugas**: Tugaskan pekerjaan kepada anggota tim
4. **Pantau Metrik**: Gunakan laporan untuk melacak kinerja
