### Where to Find It
1. Go to **Packaging** module

4. See summary at the top

## Common Tasks

### View Summary Metrics
1. Open Recap tab
2. Look at top section
3. See 4 colored cards with metrics

### Find Merged SJs
1. Scroll down to "Merged SJ Details"
2. Find the recap SJ number
3. See all merged SJ numbers listed

### Verify Consolidation
1. Check "Total SJ Recap" vs "Total SJ Merged"
2. Example: 5 recaps from 12 SJs = 2.4:1 consolidation ratio
3. Verify correct SJs were merged

### Track Volume
1. Check "Total Quantity"
2. Compare with expected shipment
3. Ensure no data loss

## FAQ

**Q: Why is the summary not showing?**  
A: The summary only shows if there are closed deliveries. Create or close a delivery first.

**Q: Can I edit the summary?**  
A: No, the summary is auto-generated. Edit individual SJs instead.

**Q: How often does it update?**  
A: Real-time - updates immediately when data changes.

**Q: Can I export the summary?**  
A: Use the individual SJ export feature for detailed reports.

**Q: What if numbers don't match?**  
A: Refresh the page. If still wrong, check data integrity.
